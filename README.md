## Instructions:

1. Place source PNGs/JPGs in folders 'art-images-source' and 'art-close-ups-source'
2. Download google doc as CSV file
3. Paste CSV data into here https://www.convertcsv.com/csv-to-json.htm
4. Paste generated JSON into art-data-source.json
5. Ensure output folders (art-images and art-close-ups) exist and are empty
6. Run batch files as appropriate
7. Copy output images and art.js across to develop branch
